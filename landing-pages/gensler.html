For a corridor serving more than 50 occupants in a high-rise B occupancy in Los Angeles, what is the minimum egress width? Does LA amend the base IBC?
Under IBC 2021 Section 1005.1, the minimum egress width is 0.2 inches per occupant for corridors - yielding 44 inches minimum for occupant loads above 50. However, LABC 2023 Section 1005.1.1 amends this requirement for high-rise B occupancies, mandating a minimum corridor width of 60 inches for corridors serving more than 50 occupants above the 55-foot threshold. The LA amendment supersedes the base IBC for this project type and must be applied.
